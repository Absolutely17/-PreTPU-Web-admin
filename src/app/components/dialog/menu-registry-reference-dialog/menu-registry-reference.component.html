Как выбрать пункты меню?

При открытии реестра пунктов меню Вам предлагается выбрать пункты меню какого языка необходимо отобразить.

Для того, чтобы сделать этот выбор необходимо в выпадающем списке нажать на один из предлагаемых языков.

В случае, если пункты меню по данному языку уже существует, то они отобразятся на экране. Иначе будет выведено предупреждение в нижней части экрана о том, что пункты меню по указанному языку не найдены. Самое время их добавить!

Что значит та или иная иконка/надпись?

Рядом с выпадающим списком доступных языков в системе расположены две кнопки:

- добавить пункт меню в корень списка (то есть, добавляемый пункт меню будет иметь 1-й уровень вложенности).
- открыть справку по пользованию функционалом реестра пунктов меню (как раз таки эту справку Вы сейчас читаете :) )

Все пункты меню представлены прямоугольными блоками с основной информацией на них. Эта информация содержит два важных параметра пункта меню, которые есть у каждого из них - название и тип пункта меню (схема следующая: Название [Тип пункта меню])

У каждого блока есть, как минимум, две кнопки. Но у пунктов меню типа "Список пунктов меню" добавляется третья.

- редактировать пункт меню. Откроется диалоговое окно в котором можно будет изменить параметры выбранного пункта.
- удалить пункт меню. По нажатию всплывет подтверждающее окно с целью избежания случайного удаления.
- (опционально, доступно лишь у "Список пунктов меню"). Добавляет пункт меню внутрь выбранного, таким образом вложенность добавляемого пункта меню будет: <вложенность пункта меню, у которого нажата эта кнопка> + 1.
Как изменить расположение пунктов меню?

Для комфортного и быстрого изменения расположения пунктов меню реализована система, позволяющая перетаскивать мышкой блоки, представляющие собой пункты меню, таким образом меняя их порядок.

Для того, чтобы начать - зажмите левую кнопку мыши на том блоке, который необходимо перенести. Пункт меню можно расположить тремя разными вариантами относительно другого. Рассмотрим каждый отдельно.

Случай, когда необходимо разместить выбранный пункт меню ниже указанного. Для этого необходимо перетащить мышкой выбранный пункт под пункт меню, ниже которого мы хотим видеть наш. При этом под указанным блоком будет отображена зеленая полоса, подсказывающая нам о том, где будет расположен выбранный пункт меню. Отпускаем левую кнопку мыши и пункты меню меняют свой порядок, согласно тому, как Вы решили их расположить.

Как изменить расположение пунктов меню?

Следующий случай - выше указанного.

В этом случае необходимо навести мышкой выбранный пункт меню над пунктом меню, выше которого необходимо расположить. По аналогии с предыдущим случаем отрисованная зеленая полоса подскажет нам о том, где будет расположен текущий выбранный пункт меню.

Как изменить расположение пунктов меню?

Последний случай - внутрь пункта с типом "Список пунктов меню".

Существуют пункты меню с типом "Список пунктов меню", которые внутри себя хранят также пункты меню. И с ними также можно взаимодействовать посредством данного функционала.

Для этого наведите перетаскиваемый пункт меню на пункт меню с типом "Список пунктов меню". При наведении его границы примут темно-синий цвет, означающий что при отпускании левой кнопки мыши перетаскиваемый элемент окажется внутри указанного.

Для извлечения пунктов меню на более высокий уровень вложенности - воспользуйтесь описанными ранее случаями.

Как редактировать содержимое пунктов меню?

Для редактирования параметров того или иного пункта меню необходимо нажать на кнопку редактирования, которая находится справа от каждого из блоков.

Некоторые параметры пунктов меню зависят от их типа, но есть и общие:

  • Название пункта меню - название, которое будет видеть пользователь
  • Тип пункта меню - тип, от которого будет зависеть дальнейшее поведение при нажатии на пункт меню
  • Изображение пункта меню - изображение, которое будет отображаться у пункта меню

Если выбранный тип пункта меню - статья - добавляются две кнопки "Выбрать статью" и "Добавить новую статью". Необходимо либо выбрать уже существующую статью, либо добавить новую, которая автоматически прикрепится к указанному пункту меню. Если у пункта меню уже выбрана статья, то по нажатию на "Выбрать статью" текущая статья будет выделена в таблице.

Если выбранный тип пункта меню - список статей - добавляется кнопка "Выбрать статьи", которая позволяет выбрать необходимые стать и прикрепить их к указанному пункту меню. Соответственно, пользователь по нажатию на данный пункт меню получит перед собой страницу со списком выбранных статей, которые будут представлены названиеим стать и её кратким содержанием. По аналогии с предыдущим типом, если статьи уже были выбраны, они отобразятся при нажатии на "Выбрать статьи"

Если выбранный тип пункта меню - ссылка, то добавляется еще одно поле "Ссылка", которая позволяет прикрепить URL по которому будет перенаправлен пользователь после нажатия на пункт меню

Если выбранный пункт меню - список пунктов меню, то ничего дополнительно здесь нет необходимости заполнять. Данный пункт позволяет положить внутрь другие пункты меню.

Если выбранный пункт меню - расписание - ничего дополнительного не требуется. Создает пункт меню, представляющий собой ссылку-переход на расписание ТПУ. При этом, если у пользователя выбрана его учебная группа, то он попадет на расписание именно своей группы.

Как сохранить изменения?

Все изменения, производимые над пунктами меню, отображаются в "История изменений", которая располагается справа от блоков с пунктами меню. В истории видны основные изменения, которые были произведены над пунктами меню.

Сохранение всех изменений производится по нажатию на кнопку в виде дискеты, которая расположена снизу справа. Кнопка появляется при первом же изменении над пунктами меню.
